# 2. Файл инициализации Project

## Рекомендованное имя файла

`00_PROJECT_INITIALIZATION.md`

## Назначение

Это главный файл, который должен быть загружен в новый Project и использоваться как первичная точка входа.

## Содержимое файла

```markdown

IPaC Core Restoration Lab

New Project Core Restoration

## Mission

Восстановить в новом ChatGPT Project ядро IPaC без переноса старого контекстного хаоса.

Главная задача:
создать устойчивое рабочее состояние Project, в котором IPaC можно продолжать развивать через артефакты, а не через бесконечный чат.

## Core Need

IPaC нужен, чтобы живой сложный смысл стал управляемым, переносимым, проверяемым и исполнимым знанием.

## Core Conflict

AI made output cheap.  
Now meaning is the bottleneck.

Перевод:
```
