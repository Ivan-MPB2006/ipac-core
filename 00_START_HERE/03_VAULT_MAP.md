## Purpose

This vault is a detachable, artifact-first knowledge base for IPaC Core Restoration Lab.
It stores active boot-level canon, supporting registers, source packages, parking lot items, decision trace, and future import staging without moving the project into MVP, Codex, RAG/Neo4j, or execution mode.
