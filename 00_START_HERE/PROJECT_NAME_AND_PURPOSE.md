# 1. Рекомендованное наименование Project

## Основное имя

**IPaC Core Restoration Lab**  
(Лаборатория восстановления ядра IPaC)

## Почему именно так

Название фиксирует правильный режим работы:

- это ещё не production project;
- это не MVP-launch project;
- это не Codex-execution project;
- это лаборатория восстановления и стабилизации ядра;
- главная задача — восстановить управляемое ядро IPaC в новом Project.

## Чего избегать в названии

Не рекомендую называть Project:

- `IPaC MVP`
- `IPaC Execution`
- `IPaC Codex`
- `IPaC Product`
- `IPaC Launch`
- `IPaC Marketing`
- `IPaC Agent System`

Почему:

такие названия преждевременно тянут мышление в execution, инструменты, публичный запуск или продуктовую рамку.
