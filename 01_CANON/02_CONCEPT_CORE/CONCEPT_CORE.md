# 01 — Concept Core Summary

## Core formula

```text
IPaC is a conceptual system for turning living complex meaning into manageable, portable, verifiable and executable knowledge.
```

Русская формула:

```text
IPaC — это концептуальная система превращения живого сложного смысла
в управляемое, переносимое, проверяемое и исполнимое знание.
```

## Key conceptual circuit

```text
Living Meaning
→ Knowledge Clump
→ Knowledge Artifact
→ Canon
→ Trace
→ Feedback / Evidence
→ Review
→ Decision
→ Controlled Rebuild
→ Updated Canon
```

Перевод:

```text
Живой смысл
→ Знаниевый сгусток
→ Знаниевый артефакт
→ Канон
→ След / трасса
→ Обратная связь / свидетельства
→ Разбор / ревью
→ Решение
→ Управляемая пересборка
→ Обновлённый канон
```

## Boundary

IPaC is not ordinary knowledge management, not project management, not AI-assisted writing, not RAG, not Codex automation and not a content factory.

## Current maturity

The concept core is accepted as `accepted-candidate`, not final canon. It must be checked against Research Findings, Open Questions and future field evidence.
