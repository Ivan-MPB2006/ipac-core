# 04 — Validation Gaps and Risk Summary

## Main validation gap

IPaC has strong internal coherence and artifact-supported structure, but it does not yet have field validation.

## Validation levels

```text
internal-dialogue
artifact-supported
research-supported
field-tested
externally-validated
```

Current status: mostly internal-dialogue / artifact-supported / research-supported.

## Key risk

Beautiful synthesis can still create false maturity.

Therefore the next phase must identify minimum validation candidates rather than expand architecture.

## Rule

No hypothesis should be promoted to field-validated until external readers, users, reviewers, or practical tests generate field evidence.
