```yaml
artifact_id: SLE-INTERNAL-ORG-INTELLIGENCE-INHERITANCE-v0.1
artifact_type: inheritance_note
status: candidate
level: upper_level
```

## 1. Суть

Блок внутренней организационной разведки относится к верхнеуровневому подходу ИГЭС / Semantic Lifecycle Engineering, а не только к IPaC.

Он должен наследоваться вниз во все специализированные ОС:

```text
NIR OS (ОС НИР)
NIOKR OS (ОС НИОКР)
Development OS (ОС разработки)
Production OS (ОС продуктива)
Modernization OS (ОС модернизации)
Retirement OS (ОС вывода из действия)
```
