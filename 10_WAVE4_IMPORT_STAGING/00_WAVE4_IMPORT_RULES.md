# Wave 4 Import Rules

## Status

Wave 4 materials are not imported into canon by default.

## Flow

```text
unreviewed inbox
→ import candidate
→ review notes
→ accepted after review OR rejected / parked
```

## Promotion rule

Nothing moves from Wave 4 staging into canon without:

1. source identification;
2. gap / relevance statement;
3. drift check;
4. review;
5. decision record;
6. parking impact check.
