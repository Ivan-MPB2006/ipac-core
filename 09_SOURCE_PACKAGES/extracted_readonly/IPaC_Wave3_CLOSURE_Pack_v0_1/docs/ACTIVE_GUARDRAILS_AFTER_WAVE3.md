# Active Guardrails After Wave 3

## Основное правило

Не переходить к MVI, пока ядро IPaC не восстановлено в новом Project.

## Stop rules

Остановить сессию, если появляется:

1. Codex / MCP / RAG / Neo4j как текущая задача.
2. Выбор MVP без review Candidate Map.
3. Execution plan до Project Core Restoration.
4. Публичные каналы как замена field design.
5. Попытка считать красивую реакцию field validation.
6. Попытка делать clean-room test внутри старого контекста.
7. Размножение документов без закрытия gap.

## Allowed now

Разрешено:

- создать новый Project;
- импортировать Starter Pack v0.2;
- провести boot review;
- проверить, что Project восстанавливает координаты;
- уточнить open questions;
- обновить parking lot.

## Not allowed now

Не разрешено:

- запускать MVI;
- выбирать окончательный MVP;
- делать test design;
- делать external reviewer loop;
- подключать Codex;
- проектировать full stack.
