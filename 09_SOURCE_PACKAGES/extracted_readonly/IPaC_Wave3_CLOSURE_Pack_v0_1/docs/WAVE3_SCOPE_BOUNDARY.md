# Wave 3 Scope Boundary

## Что входило в Wave 3

Wave 3 должна была восстановить из старой работы:

- кандидаты MVP / MVI;
- какие гипотезы они проверяют;
- какие открытые вопросы затрагивают;
- какое минимальное field evidence могут дать;
- какие варианты преждевременны.

## Что не входило в Wave 3

Wave 3 не должна была:

- запускать MVP;
- выбирать окончательный MVI;
- проектировать тест;
- создавать execution plan;
- начинать внешнюю коммуникацию;
- подключать Codex;
- строить техническую архитектуру.

## Итоговая граница

Phase 7 — внутри scope.  
Phase 7R — на границе scope, сохранить как draft.  
Phase 7S / Phase 7T — за пределами текущего scope, припарковать.
