# 00 — Knowledge Recovery Overview

## Purpose

Wave 2 converts recovered core discipline into a structured content layer:

```text
Need + Architecture + Hypotheses + Lessons
        ↓
Concept + Research + Open Questions
        ↓
Knowledge Recovery Pack
```

## Main result

IPaC is now represented not as a long chat, but as a linked set of portable artifacts:

- concept core;
- glossary;
- research map;
- source limitation register;
- research gaps;
- open questions;
- contradictions;
- assumptions;
- validation gaps;
- parked questions.

## Package status

`accepted-candidate`, not `accepted-canon`.

The package is strong enough to initialize the next Project layer, but it still needs validation through external review, field evidence and MVP/MVI experiments.
