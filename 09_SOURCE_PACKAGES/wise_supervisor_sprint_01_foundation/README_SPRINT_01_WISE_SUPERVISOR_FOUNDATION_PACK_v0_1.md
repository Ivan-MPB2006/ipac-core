# README — Sprint 01 Wise Supervisor Foundation Pack

## Как использовать

1. Распаковать ZIP (ZIP-пакет) в корень Vault (хранилища).
2. Проверить список файлов по MANIFEST (манифесту).
3. Выполнить Git (Гит)-проводку только targeted add (точечным добавлением) по списку файлов.
4. Не использовать `git add .`.

## Смысловой квант

Этот Sprint Pack (спринтовый пакет) является одним смысловым квантом: он оформляет доктрину, процесс, правила и прецедентную базу Мудрого Supervisor (супервизора).
