# README — IPaC Thread-to-Scene Agentic Work Routing Pack v0.1
## README пакета перехода от треда к сценовой агентной работе

Статус: **candidate (кандидат)**, **not canon (не канон)**.

Этот пакет выгружает стратегическую перенастройку из Thread (треда) и раскладывает её по реестрам Obsidian Vault (хранилища Obsidian).

Главная формула:

```text
Мы не строим идеальный Browser Thread (браузерный тред).
Мы строим механизм выхода из Thread (треда)
в Scene-based Agentic Work (сценовую агентную работу).
```
