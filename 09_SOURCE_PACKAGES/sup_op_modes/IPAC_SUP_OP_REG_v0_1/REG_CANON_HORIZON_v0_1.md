# Canon Lifecycle Register (регистр жизненного цикла канона)
## Canon as Moving Horizon (канон как движущийся горизонт)

```yaml
status: candidate (кандидат)
canon_status: not_canon (не канон)
promotion_authorized: false
git_commit_authorized: false
human_approval_required: true
```

Canon (канон) не является конечной станцией. Canon (канон) является Moving Horizon (движущимся горизонтом), Guiding Star (путеводной звездой) и ориентиром для улучшения Candidate (кандидата).

```text
Weak Signal (слабый сигнал)
→ Candidate (кандидат)
→ Strengthened Candidate (усиленный кандидат)
→ Tested Candidate (проверенный кандидат)
→ Working Rule (рабочее правило)
→ Working Canon with Scope (рабочий канон с областью применения)
→ Canon Under Watch (канон под наблюдением)
→ Superseded Canon (заменённый канон)
→ Historical Source / Factographic Canon
   (исторический источник / фактографический канон)
```

Anti-pattern (антипаттерн): Canon Fetishization (фетишизация канона).
