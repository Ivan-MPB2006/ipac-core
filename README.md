# IPaC Obsidian Vault Structure v0.1 Pack

## Status

Detachable ZIP package for Obsidian / Git Knowledge Base preparation.

This is not:

- MVP execution
- MVI test design
- Codex implementation
- RAG / Neo4j design
- automation setup
- public-channel execution

## Contents

- Obsidian Vault folder structure
- Core Canon placement scaffold
- Canon status and acceptance-note files
- Centralized register folders
- Parking Lot section
- Trace / Decision history section
- Source package raw ZIPs and readonly extracted copies
- Future Wave 4 import staging area
- Templates for future manual review and decision discipline

## Use

Open the root folder `IPaC_Obsidian_Vault_Structure_v0_1_Pack` as an Obsidian Vault.

Recommended first file:

`00_START_HERE/00_PROJECT_INITIALIZATION.md`

Recommended navigation file:

`00_START_HERE/03_VAULT_MAP.md`

## Boundary

The package preserves the accepted boot-level canon frame and source materials, but it does not declare any parked execution item as active canon.
